# AINITY - Universal Learning & Innovation Platform

AINITY is a premium, full-stack educational platform designed to empower people across all sectors and geographies to upskill and turn their ambitious ideas into reality.

## 🚀 Vision
Our mission is to ensure that background and geography are never barriers to growth. From tech hubs to rural villages, AINITY provides the tools for anyone to drive innovation in their own domain.

## 🛠️ Technology Stack
- **Frontend:** Next.js (App Router), Tailwind CSS, Framer Motion, Lucide Icons.
- **Backend:** FastAPI (Python), SMTP for notifications, Secure Admin Dashboard.
- **Security:** Environment-based configuration, header-based admin authentication.

## 📂 Project Structure
- `/frontend`: Next.js web application.
- `/backend`: FastAPI server and data management.

## ⚡ Quick Start

### Backend
1. `cd backend`
2. `python -m venv venv`
3. `source venv/bin/activate`
4. `pip install fastapi uvicorn python-dotenv`
5. `uvicorn main:app --reload`

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm run dev`
